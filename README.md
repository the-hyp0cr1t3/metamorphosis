## Build Instructions

Build using [CMake](https://cmake.org/). For more details, see [BUILDING.md](BUILDING.md).

```bash
git clone git@github.com:the-hyp0cr1t3/rasterization-algos.git
cd rasterization-algos

# Build
mkdir build
cd build
cmake ..
make

# Install
sudo make install
```
